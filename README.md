# 赢了么

基于 `服了么.md` 落地的正式网页游戏原型，不再是静态演示页。

## 技术栈

- `Phaser 4`
- `TypeScript`
- `Vite`
- `GitHub Pages`

## 当前内容

- 标题场景 + 战斗场景
- 四资源战斗：体面 / 气势 / 舆论 / 破防
- 40 张测试牌
- 轻响应系统
- 3 个普通敌人 + 1 个 Boss
- 2 个场地
- 战后加卡与整条 run 闭环

## 本地运行

```bash
npm install
npm run dev
```

## 验证

```bash
npm run build
npm run smoke
```
